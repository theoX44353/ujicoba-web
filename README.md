### Langkah-langkah Instalasi
1. Clone repositori ini:
    ```bash
    git clone https://github.com/theoX44353/ujicoba-web.git
    cd ujicoba-web
    ```

2. Install dependensi Python:
    ```bash
    pip install -r requirements.txt
    ```

3. Install dependensi Node.js:
    ```bash
    cd static/js
    npm install
    ```
